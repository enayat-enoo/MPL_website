# MPL Website

This project backend is deployed on Render’s free tier. Since the server goes to sleep after 15 minutes of inactivity, UptimeRobot is temporarily configured to keep it active for demonstration purposes. A paid plan or dedicated VPS would be preferred for production use.

## Live Demo

Check out the live frontend: [MPL Website](https://mpl-website.vercel.app)

Backend API (hosted on Render): [MPL API](https://mpl-backend-ct21.onrender.com)
